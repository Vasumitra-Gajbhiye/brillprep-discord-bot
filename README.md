# BrillPrep Discord Bot

The official Discord bot for BrillPrep.

Students can practice A Level and IGCSE questions directly inside Discord through interactive MCQs, explanations, streaks, leaderboards, and more.

## Features

- Practice questions
- Instant answer checking
- Explanations
- Streaks and XP (planned)
- Performance tracking (planned)
- Integration with BrillPrep API

## Tech Stack

- TypeScript
- Discord.js
- Node.js
- BrillPrep API

## Development

```bash
npm install
npm run dev
```

Create a `.env` file:

```env
DISCORD_TOKEN=your_token
CLIENT_ID=your_client_id
```
